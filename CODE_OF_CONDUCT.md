# Code of Conduct

## Our Commitment

As a university software engineering project (TFG), we are committed to maintaining a
professional and respectful environment for contributors, reviewers, and users.

## Standards

### Expected Behavior

- Constructive feedback only
- Respect for different perspectives and experience levels
- Clear, documented code and commit messages
- Timely response to pull requests and issues

### Unacceptable Behavior

- Personal attacks or harassment
- Dismissing contributions based on skill level
- Sharing private information of others
- Any form of discrimination

## Enforcement

Reports of unacceptable behavior may be directed to the project maintainer:

**Adrián Martínez** — Universidad de Oviedo (UO295454)
